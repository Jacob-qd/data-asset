# 隐私计算二级页面需求文档

## 概述

隐私计算模块包含6个二级页面：任务中心、隐私求交(PSI)、隐匿查询(PIR)、联合统计、联合SQL、联合建模。每个页面需要具备完整的业务功能，包括任务列表展示、新建任务、任务详情查看、任务操作等。

---

## 1. 隐私计算任务中心 (/privacy/tasks)

### 1.1 页面定位
汇总视图，展示所有类型的隐私计算任务，提供全局统计和快速入口。

### 1.2 功能需求

#### 统计卡片区域
- **总任务数**：当前用户参与的所有任务总数
- **各类型任务数量**：PSI / PIR / Stats / SQL / Modeling 分别的数量
- **今日执行数**：今日执行的任务数量
- **成功率**：近7天任务执行成功率

#### 任务列表
- 表格列：序号、任务名称、任务类型、业务标签、算法类型、参与方状态、执行状态、创建时间、操作
- 支持按任务类型筛选
- 支持按状态筛选
- 支持搜索任务名称

#### 快速入口
- 每个类型一个快捷新建按钮
- 点击跳转对应类型的新建任务页面

#### 最近执行记录
- 展示最近5条执行记录
- 包含：任务名、类型、状态、执行时间

---

## 2. 隐私求交 (PSI) (/privacy/tasks/psi)

### 2.1 业务定义
隐私求交（Private Set Intersection）允许多个参与方在不泄露各自数据的前提下，计算数据集合的交集。

### 2.2 功能需求

#### 任务列表
- 表格列：序号、任务名称、求交键字段、参与方数量、数据量级、执行状态、创建时间、操作
- 筛选：按状态、按参与方数量
- 搜索：任务名称

#### 新建任务对话框
**基本信息**：
- 任务名称（必填）
- 业务标签（下拉选择）
- 描述（文本域）

**参与方配置**：
- 添加参与方（至少2个）
- 每个参与方：节点名称、数据表选择、求交键字段选择
- 支持删除参与方

**资源配置**：
- 算力资源选择
- 使用期限
- 允许使用次数

#### 任务详情
- 基本信息展示
- 参与方列表及状态
- 求交结果统计（交集数量、各参与方贡献量）
- 结果下载按钮
- 执行日志

#### 操作按钮
- 执行、暂停、删除、查看详情、查看日志

---

## 3. 隐匿查询 (PIR) (/privacy/tasks/pir)

### 3.1 业务定义
隐匿查询（Private Information Retrieval）允许查询方在不暴露查询内容的前提下，从被查询方获取数据。

### 3.2 功能需求

#### 任务列表
- 表格列：序号、任务名称、查询类型、被查询方、查询条件、响应状态、创建时间、操作
- 筛选：按查询类型、按响应状态
- 搜索：任务名称

#### 新建任务对话框
**基本信息**：
- 任务名称（必填）
- 业务标签
- 描述

**查询配置**：
- 查询方选择
- 被查询方选择
- 查询类型：关键词查询 / 范围查询 / 条件查询
- 查询条件设置（根据类型动态变化）
- 结果返回格式：单条 / 多条 / 聚合

**资源配置**：
- 算力资源选择
- 使用期限
- 允许使用次数

#### 任务详情
- 基本信息展示
- 查询条件展示
- 查询结果（脱敏展示）
- 响应日志
- 结果导出

#### 操作按钮
- 执行、暂停、删除、查看详情、查看日志

---

## 4. 联合统计 (/privacy/tasks/stats)

### 4.1 业务定义
联合统计允许多个参与方在不泄露原始数据的前提下，共同计算统计指标。

### 4.2 功能需求

#### 任务列表
- 表格列：序号、任务名称、统计类型、统计维度、参与方数量、执行状态、创建时间、操作
- 筛选：按统计类型、按状态
- 搜索：任务名称

#### 新建任务对话框
**基本信息**：
- 任务名称（必填）
- 业务标签
- 描述

**统计配置**：
- 统计类型选择：求和 / 均值 / 方差 / 标准差 / 分位数 / 计数 / 最值
- 统计字段选择
- 分组字段选择（可选）
- 筛选条件（可选）

**参与方配置**：
- 添加参与方
- 每个参与方：数据表、统计字段映射

**资源配置**：
- 算力资源选择
- 使用期限

#### 任务详情
- 基本信息展示
- 统计结果展示（表格/图表）
- 各参与方贡献统计
- 结果导出

#### 操作按钮
- 执行、暂停、删除、查看详情、查看日志

---

## 5. 联合SQL (/privacy/tasks/sql)

### 5.1 业务定义
联合SQL允许多个参与方在不泄露原始数据的前提下，执行跨方的SQL查询。

### 5.2 功能需求

#### 任务列表
- 表格列：序号、任务名称、SQL预览（前50字符）、参与方数量、执行状态、创建时间、操作
- 筛选：按状态
- 搜索：任务名称、SQL内容

#### 新建任务对话框
**基本信息**：
- 任务名称（必填）
- 业务标签
- 描述

**SQL配置**：
- SQL编辑器（支持语法高亮）
- 参与方数据表选择
- 字段自动补全提示
- SQL验证按钮

**查询参数**：
- 参数化查询配置（可选）
- 结果行数限制
- 结果排序配置

**参与方配置**：
- 添加参与方
- 每个参与方：数据表、字段映射

#### 任务详情
- 基本信息展示
- 完整SQL展示
- 查询结果表格（支持分页）
- 执行计划展示
- 结果导出（CSV/JSON）

#### 操作按钮
- 执行、暂停、删除、查看详情、查看日志

---

## 6. 联合建模 (/privacy/tasks/modeling)

### 6.1 业务定义
联合建模（联邦学习）允许多个参与方在不共享原始数据的前提下，共同训练机器学习模型。

### 6.2 功能需求

#### 任务列表
- 表格列：序号、任务名称、算法类型、模型名称、训练进度、执行状态、创建时间、操作
- 筛选：按算法类型、按状态
- 搜索：任务名称

#### 新建任务对话框
**基本信息**：
- 任务名称（必填）
- 业务标签
- 描述

**算法配置**：
- 算法类型选择：
  - 纵向联邦：逻辑回归、XGBoost、神经网络
  - 横向联邦：FedAvg、FedProx、SCAFFOLD
- 模型名称
- 模型版本

**数据配置**：
- 添加参与方
- 每个参与方：
  - 数据表选择
  - 特征字段选择
  - 标签字段选择（仅发起方）
  - 样本ID字段（用于对齐）

**超参数配置**：
- 学习率
- 迭代轮数
- 批量大小
- 正则化参数
- 早停配置

**训练配置**：
- 训练/验证/测试集比例
- 评估指标选择
- 模型保存策略

#### 任务详情
- 基本信息展示
- 训练进度展示（进度条/曲线图）
- 模型评估指标（准确率、AUC、F1等）
- 训练损失曲线
- 各参与方贡献统计
- 模型下载
- 模型发布（发布到模型管理中心）

#### 操作按钮
- 执行训练、暂停、删除、查看详情、查看日志
- 可视化建模入口（跳转到 VisualModelingIDE）

## 阶段1开发计划

### 目标
为每个任务类型创建独立的页面组件，包含：
1. 独立的任务列表页面（带类型专属字段）
2. 新建任务对话框（带类型专属表单）
3. 更新路由配置
4. 更新Mock数据以支持新字段

### 页面清单
1. `PSITaskPage.tsx` - 隐私求交
2. `PIRTaskPage.tsx` - 隐匿查询
3. `StatsTaskPage.tsx` - 联合统计
4. `SQLTaskPage.tsx` - 联合SQL
5. `ModelingTaskPage.tsx` - 联合建模
6. 更新 `PrivacyTaskCenter.tsx` - 任务中心（汇总视图）

### 数据模型扩展
扩展 `PrivacyTask` 接口，增加各类型专属字段。

### Mock数据
为每种类型创建独立的Mock数据集，包含丰富的示例数据。
