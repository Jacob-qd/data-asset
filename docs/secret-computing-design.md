# 密态计算模块设计文档

## 一、需求梳理

原始需求包含 120+ 个功能点，按业务流程整合为以下模块：

| 原始分类 | 功能数量 | 整合后模块 |
|---------|---------|-----------|
| 项目元数据/成员/数据授权/统计 | 9 | 密态项目 |
| 任务元数据/配置/执行/结果/历史 | 15 | 密态任务 |
| PSI任务 | 5 | 隐私求交 |
| PIR任务 | 5 | 隐匿查询 |
| 联合统计 | 6 | 联合统计 |
| 联邦学习 | 10 | 联邦学习 |
| 参与方信息/密钥 | 4 | 参与方管理 |
| 存证信息/审计日志/操作追溯 | 7 | 审计存证 |
| 监控指标/告警规则/Dashboard | 7 | 监控告警 |
| 资源池/配额/调度 | 5 | 资源管理 |
| 数据质量/数据血缘 | 7 | 数据治理 |
| 密钥生命周期/证书管理 | 6 | 密钥证书 |
| 网络拓扑/通信协议 | 3 | 网络通信 |
| 模型仓库/版本控制 | 3 | 模型管理 |
| MPC/TEE/同态加密 | 8 | 密码学引擎 |
| 合规策略/检查任务 | 4 | 合规管理 |
| REST API/SDK/Webhook | 4 | 接口管理 |
| 数据服务目录/接口/版本 | 9 | 服务管理 |

## 二、菜单整合方案

按**功能内聚、流程导向、扁平优先**原则，将 120+ 功能点整合为 **13 个二级菜单项**：

### 隐私计算 > 密态计算 子菜单

1. **密态项目** `/secret/projects`
   - 项目列表（创建/编辑/删除/冻结/解冻）
   - 项目详情（元数据、成员、数据授权）
   - 统计报表

2. **密态任务** `/secret/tasks`
   - 任务列表（创建/编辑/删除/复制）
   - 执行控制（提交/取消/重试/暂停/恢复）
   - 结果管理（查看/导出/下载）
   - 执行历史、进度、日志

3. **隐私求交(PSI)** `/secret/psi`
   - 任务配置（求交字段、模式、输出方式）
   - 任务创建与执行
   - 结果查看/导出
   - 执行统计

4. **隐匿查询(PIR)** `/secret/pir`
   - 任务配置（查询字段、条件、索引）
   - 任务创建与执行
   - 索引构建
   - 结果查看、执行统计

5. **联合统计** `/secret/joint-stats`
   - 任务配置（指标、分组、聚合方式）
   - 任务创建与执行
   - 结果查看/导出/可视化
   - 执行统计

6. **联邦学习** `/secret/federated`
   - 模型元数据管理
   - 训练配置（轮数、学习率、早停策略）
   - 训练/评估/发布/下线
   - 评估指标管理
   - 训练过程监控

7. **参与方管理** `/secret/parties`
   - 参与方信息管理
   - 密钥管理

8. **密钥证书** `/secret/keys`
   - 密钥生命周期管理
   - 密钥权限/备份恢复
   - 证书生命周期管理
   - 证书链/吊销列表

9. **审计存证** `/secret/audit`
   - 存证配置与记录查询
   - 存证验证
   - 审计日志
   - 操作追溯与报表导出

10. **监控告警** `/secret/monitor`
    - 监控指标配置
    - 告警规则配置
    - 监控 Dashboard
    - 告警列表查询与处理
    - 监控报表导出

11. **资源管理** `/secret/resources`
    - 资源池管理
    - 资源配额策略
    - 资源使用记录
    - 资源预留与调度策略

12. **数据治理** `/secret/data-governance`
    - 数据质量规则库/检查任务/报告/问题
    - 数据血缘元数据/关系图谱/影响分析

13. **服务管理** `/secret/services`
    - API/SDK/Webhook/消息队列接口
    - 数据服务目录/接口/版本
    - 服务监控/限流/熔断/降级
    - 服务认证
